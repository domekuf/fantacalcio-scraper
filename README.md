# fantacalcio scraper
Scrape Serie A soccer player's stats!

## Install
npm install domekuf/fantacalcio-scraper

## Get a template
curl https://raw.githubusercontent.com/domekuf/fantacalcio-scraper/main/template.csv -o template.csv

## Run
npx fcs -i template.csv
